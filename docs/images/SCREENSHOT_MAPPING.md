## Quick Replace Commands

After you've saved all images, you can use find/replace in your editor:

**VS Code / IDE:**
1. Open find/replace (Cmd/Ctrl + H)
2. Enable regex mode
3. Find: `\[Screenshot: (.*?)\]`
4. Replace: `![$1](images/[filename].png)`
   (You'll need to do these individually with correct filenames)

Or use these specific replacements document by document.
