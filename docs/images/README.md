# Screenshots

This folder contains all screenshot images used in the documentation.

## Screenshot Naming Convention

Use descriptive, kebab-case names that match the placeholder descriptions:

- `main-interface-tabs.png` - Main interface showing three tabs
- `upload-drag-drop.png` - File upload area with drag and drop
- `file-browser-dialog.png` - File browser dialog window
- `file-selected.png` - Interface showing selected file
- `process-preview-button.png` - Process & Preview button
- `processing-indicator.png` - Processing spinner/indicator
- `preview-table.png` - Data preview table
- `success-message.png` - Success message card
- `error-message.png` - Error message card
- `warning-message.png` - Warning message card
- And so on...

## Image Guidelines

- **Format**: PNG (preferred) or JPG
- **Resolution**: 1920x1080 or similar
- **File Size**: Keep under 500KB when possible (optimize if needed)
- **Clarity**: Ensure text and UI elements are clearly readable

## Taking Screenshots

### Mac
- Full screen: `Cmd + Shift + 3`
- Selected area: `Cmd + Shift + 4`
- Window: `Cmd + Shift + 4` then press `Space`

### Windows
- Full screen: `PrtScn` or `Windows + Shift + S`
- Snipping Tool: Search for "Snipping Tool" in Start menu
- Selected area: `Windows + Shift + S`

## Embedding in Markdown

Replace placeholder text like:
```markdown
[Screenshot: Upload interface]
```

With:
```markdown
![Upload interface](images/upload-interface.png)
```
